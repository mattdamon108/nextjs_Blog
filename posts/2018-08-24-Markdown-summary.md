### a Note for future me about Markdown syntax

## Headers

```code
# H1
## H2
### H3
#### H4
##### H5
###### H6
```

<br/>

## Blockquote

```code
> This is a blockquote
```

> This is a blockquote

<br/>

## List

<br/>

### 1) Ordered

```code
1. first
2. second
3. third
```

1. first
2. second
3. thrid

<br/>

### 2) UnOrdered

```code
- Red
  - Green
    - Blue

or '+', '-', '='
```

- Red
  - Green
    - Blue

<br/>

## 6. Link

```code
<https://www.google.com>
[Google](https://www.google.com)
```

<https://www.google.com><br />

<br />

## 7. Text Style

```code
*italic* or _italic_
**bold** or __bold__
***italic bold*** or ___italic bold___
<u>underline</u>
~~canceline~~
```

_italic_<br />
**bold**<br />
**_italic bold_**<br />
<u>underline</u><br />
~~canceline~~

<br />

